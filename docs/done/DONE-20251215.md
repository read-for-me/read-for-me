## Phase 1: 프론트엔드 UI 및 백엔드 셋업 (시작)

### 프론트엔드 UI 껍데기 (Visuals)

- [x] Next.js App Router 설정 (`app/layout.tsx`, `app/page.tsx`)
- [x] 반응형 레이아웃 (데스크탑 2단 / 모바일 1단)
- [x] **주요 컴포넌트 구현:**
  - `InputArea`: URL 입력 UI 및 유효성 검사
  - `SourcePanel`: 링크 미리보기 카드 UI (더미 데이터)
  - `InsightCard`: 3줄 요약 UI 및 로딩 스켈레톤 (더미 데이터)
  - `AudioPlayerCard`: 오디오 플레이어 컨트롤, 배속, 진행 바 UI (더미 데이터)

**생성된 컴포넌트:**

| 컴포넌트          | 파일 경로                              | 역할                                       |
| ----------------- | -------------------------------------- | ------------------------------------------ |
| `InputArea`       | `frontend/components/input-area.tsx`   | URL 입력 폼, 유효성 검사, 제출 버튼        |
| `SourcePanel`     | `frontend/components/source-panel.tsx` | 크롤링된 소스 정보 카드 (제목, 미리보기)   |
| `InsightCard`     | `frontend/components/insight-card.tsx` | AI 요약 결과 표시, 로딩 스켈레톤           |
| `AudioPlayerCard` | `frontend/components/audio-player-card.tsx` | 오디오 플레이어 컨트롤 (재생, 배속, 진행바) |

**레이아웃 구조:**

```
┌─────────────────────────────────────────────────────┐
│                    Header                           │
├─────────────────────────────────────────────────────┤
│                   InputArea                         │
│              (URL 입력 + 제출 버튼)                  │
├──────────────────────┬──────────────────────────────┤
│                      │                              │
│     SourcePanel      │      InsightCard             │
│   (소스 미리보기)     │     (AI 요약 결과)           │
│                      │                              │
├──────────────────────┴──────────────────────────────┤
│                  AudioPlayerCard                    │
│              (오디오 플레이어 컨트롤)                 │
└─────────────────────────────────────────────────────┘
```

**반응형 레이아웃:**

- **데스크탑 (md 이상)**: 2단 레이아웃 - SourcePanel(좌) | InsightCard(우)
- **모바일 (sm 이하)**: 1단 레이아웃 - 세로 스택

**`InputArea` 컴포넌트:**

```tsx
// URL 입력 및 유효성 검사
<div className="input-area">
  <input 
    type="url" 
    placeholder="URL을 입력하세요..."
    value={url}
    onChange={handleUrlChange}
  />
  <button onClick={handleSubmit} disabled={!isValidUrl}>
    분석하기
  </button>
</div>
```

**`SourcePanel` 컴포넌트:**

```tsx
// 크롤링된 소스 정보 표시
<Card>
  <CardHeader>
    <img src={ogImage} alt={title} />
    <CardTitle>{title}</CardTitle>
  </CardHeader>
  <CardContent>
    <p>{previewText}</p>
    <a href={url}>원문 보기</a>
  </CardContent>
</Card>
```

**`InsightCard` 컴포넌트:**

```tsx
// AI 요약 결과 표시 (3줄 요약)
<Card>
  <CardHeader>
    <CardTitle>AI 요약</CardTitle>
  </CardHeader>
  <CardContent>
    {isLoading ? (
      <Skeleton />
    ) : (
      <ul>
        {bulletPoints.map((point, i) => (
          <li key={i}>{point}</li>
        ))}
      </ul>
    )}
  </CardContent>
</Card>
```

**`AudioPlayerCard` 컴포넌트:**

```tsx
// 오디오 플레이어 컨트롤
<Card>
  <div className="controls">
    <button onClick={togglePlay}>
      {isPlaying ? <PauseIcon /> : <PlayIcon />}
    </button>
    <Slider value={currentTime} max={duration} onChange={seek} />
    <select value={playbackRate} onChange={setSpeed}>
      <option value={1}>1x</option>
      <option value={1.5}>1.5x</option>
      <option value={2}>2x</option>
    </select>
  </div>
</Card>
```

**사용된 UI 라이브러리:**

- `shadcn/ui`: Card, Button, Input, Slider, Skeleton 등
- `lucide-react`: 아이콘 (Play, Pause, Volume 등)
- `tailwindcss`: 스타일링
